# Atlax MindDock

Atlax MindDock 是一款本地优先、结构化优先、输入自由的知识工作流产品。

## Vision
连接灵感输入、知识沉淀、结构管理与 AI 协作，让个人知识系统真正可持续生长。

## Repository Structure

```text
apps/

```

## Branch Strategy

- main：稳定主分支
- develop：日常开发集成分支
- feature/*：功能开发
- fix/*：缺陷修复
- refactor/*：重构优化
- docs/*：文档更新
